# Facial expression recognition Рђћ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў

рЃљрЃЏ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃбрЃўрЃърЃўрЃА рЃюрЃћрЃўрЃарЃЮрЃюрЃБрЃџрЃў рЃЦрЃАрЃћрЃџрЃћрЃЉрЃў facial expression recognition рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃљ рЃўрЃДрЃЮ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃўрЃА рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃљ, рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃфрЃЋрЃџрЃљ, рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃўрЃА рЃљрЃюрЃљрЃџрЃўрЃќрЃў рЃЊрЃљ рЃДрЃЋрЃћрЃџрЃљрЃцрЃарЃўрЃА рЃџрЃЮрЃњрЃўрЃарЃћрЃЉрЃљ Weights & Biases (Wandb)-рЃќрЃћ.

---

## ­Ъћг рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃў:

### 1№ИЈРЃБ SimpleCNN РђЊ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў 1

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃЮрЃа-рЃерЃарЃўрЃљрЃюрЃў рЃЎрЃЮрЃюрЃЋрЃЮрЃџрЃБрЃфрЃўрЃБрЃарЃў рЃюрЃћрЃўрЃарЃЮрЃюрЃБрЃџрЃў рЃЦрЃАрЃћрЃџрЃў.
- **рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў**:
  - рЃАрЃљрЃбрЃарЃћрЃюрЃўрЃюрЃњрЃЮ рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~65%
  - рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~47%
- **рЃљрЃюрЃљрЃџрЃўрЃќрЃў**:
  - **Underfitting** Рђћ рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃАрЃЌрЃЋрЃўрЃА.
  - рЃЏрЃфрЃўрЃарЃћ рЃќрЃЮрЃЏрЃўрЃА рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃљрЃа рЃўрЃФрЃџрЃћрЃЋрЃљ рЃАрЃљрЃ«рЃўрЃА рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃўрЃА рЃАрЃљрЃЎрЃЏрЃљрЃарЃўрЃАрЃљрЃЊ рЃљрЃЏрЃЮрЃфрЃюрЃЮрЃЉрЃљрЃА.

---

### 2№ИЈРЃБ DeeperCNN РђЊ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў 2

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃБрЃџрЃўрЃљ рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ Conv-рЃерЃарЃћрЃАрЃў, BatchNorm рЃЊрЃљ Dropout рЃарЃћрЃњрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљ.
- **рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў**:
  - рЃАрЃљрЃбрЃарЃћрЃюрЃўрЃюрЃњрЃЮ рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~85%
  - рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~60%
- **рЃљрЃюрЃљрЃџрЃўрЃќрЃў**:
  - **рЃќрЃарЃЊрЃљ рЃАрЃўрЃќрЃБрЃАрЃбрЃћрЃерЃў**, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЏрЃљрЃўрЃюрЃф рЃљрЃарЃўрЃА overfitting-рЃў.
  - рЃарЃћрЃњрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљ рЃњрЃљрЃарЃЎрЃЋрЃћрЃБрЃџрЃгрЃўрЃџрЃљрЃЊ рЃЊрЃљрЃћрЃ«рЃЏрЃљрЃарЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃљрЃГрЃўрЃарЃЮ рЃўрЃДрЃЮ рЃБрЃцрЃарЃЮ рЃФрЃџрЃўрЃћрЃарЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ.

---

### 3№ИЈРЃБ ResNet РђЊ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў 3

- **рЃљрЃдрЃгрЃћрЃарЃљ**: ResNet18 рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃерЃћрЃфрЃЋрЃџрЃўрЃџрЃў Conv1-рЃќрЃћ (1 рЃљрЃарЃ«рЃўрЃљрЃюрЃў input), рЃЎрЃџрЃљрЃАрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљрЃќрЃћ рЃЏрЃЮрЃарЃњрЃћрЃЉрЃБрЃџрЃў.
- **рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў**:
  - рЃАрЃљрЃбрЃарЃћрЃюрЃўрЃюрЃњрЃЮ рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~95%
  - рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~66РђЊ68%
- **рЃљрЃюрЃљрЃџрЃўрЃќрЃў**:
  - рЃЎрЃљрЃарЃњрЃў рЃЉрЃљрЃџрЃљрЃюрЃАрЃў рЃгрЃЋрЃарЃЌрЃюрЃљрЃАрЃљ рЃЊрЃљ рЃњрЃћрЃюрЃћрЃарЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃерЃЮрЃарЃўрЃА.
  - Residual рЃЉрЃџрЃЮрЃЎрЃћрЃЉрЃў рЃБрЃЎрЃћрЃЌрЃћрЃАрЃљрЃЊ рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃћрЃю рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃа рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃА.
  - рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌ рЃЊрЃљрЃЉрЃљрЃџрЃў overfitting.

---

### 4№ИЈРЃБ Vision Transformer (ViT) РђЊ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў 4

- **рЃљрЃдрЃгрЃћрЃарЃљ**: рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃўрЃљ Huggingface-рЃўрЃА `ViTForImageClassification`, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃгрЃЋрЃарЃЌрЃюрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ рЃАрЃБрЃАрЃбрЃљрЃЊ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃЊрЃљ, рЃгрЃЋрЃарЃЌрЃюрЃўрЃЌ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЊрЃљ.
- **рЃерЃћрЃЊрЃћрЃњрЃћрЃЉрЃў**:
  - рЃАрЃљрЃбрЃарЃћрЃюрЃўрЃюрЃњрЃЮ рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~99%
  - рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃќрЃБрЃАрЃбрЃћ: ~62РђЊ65%
- **рЃљрЃюрЃљрЃџрЃўрЃќрЃў**:
  - ViT рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃЉрЃћрЃЋрЃа рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃАрЃљ рЃЊрЃљ рЃарЃћрЃњрЃБрЃџрЃљрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃА.
  - рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃў рЃЊрЃарЃЮ рЃњрЃљрЃфрЃўрЃџрЃћрЃЉрЃўрЃЌ рЃЊрЃўрЃЊрЃўрЃљ.
  - рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃљ pretraining-рЃўрЃЌ.

---

## ­ЪЊѕ рЃљрЃюрЃљрЃџрЃўрЃќрЃў Wandb-рЃерЃў

- рЃДрЃЋрЃћрЃџрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў рЃљрЃарЃўрЃА рЃЊрЃљрЃџрЃЮрЃњрЃўрЃџрЃў [Wandb]([https://wandb.ai/](https://wandb.ai/tekla-tsutskha-free-university-of-tbilisi-/facial-expression-recognition?nw=nwuserteklatsutskha)) рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃљрЃќрЃћ.
- рЃЌрЃўрЃЌрЃЮ рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃџрЃўрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў Run.
- рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃњрЃарЃљрЃцрЃўрЃЎрЃћрЃЉрЃў:
  - Accuracy vs Epoch
  - Loss vs Epoch
  - Overfitting (Train-Val Gap)

---

## РюЁ рЃЊрЃљрЃАрЃЎрЃЋрЃюрЃљ

- рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃў рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ рЃўрЃДрЃЮ **ResNet18**, рЃарЃЮрЃњрЃЮрЃарЃф рЃЉрЃљрЃџрЃљрЃюрЃАрЃўрЃА, рЃўрЃАрЃћ рЃерЃћрЃЊрЃћрЃњрЃўрЃА рЃЏрЃ«рЃарЃўрЃЋ.
- ViT рЃЎрЃљрЃарЃњрЃў рЃърЃћрЃарЃАрЃърЃћрЃЦрЃбрЃўрЃЋрЃљрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА рЃЊрЃўрЃЊрЃў рЃарЃћрЃАрЃБрЃарЃАрЃћрЃЉрЃў.
- Model Selection-рЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЊрЃљ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў рЃЎрЃљрЃарЃњрЃљрЃЊ рЃљрЃЕрЃЋрЃћрЃюрЃћрЃЉрЃА underfitting-рЃАрЃљ рЃЊрЃљ overfitting-рЃА.
